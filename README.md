# Valorant 채팅 번역기

발로란트에서 두 가지 방식으로 채팅을 번역해 주는 툴입니다.

- **전송 번역**: 내가 채팅창에 입력한 텍스트를 단축키 한 번으로 다른 언어로 교체 후 전송
- **읽기 번역**: 화면 왼쪽 하단 채팅을 단축키 한 번으로 캡처 → OCR → 한국어 번역 오버레이 표시

Google Translate 기반으로 **API 키 없이** 무료로 사용할 수 있습니다.

## 다운로드 (exe)

별도 설치 없이 바로 실행하려면 [Releases](../../releases/latest) 페이지에서 `ValorantTranslator.exe`를 다운로드하세요.

> 관리자 권한이 필요합니다 (키보드 후킹을 위해 자동 요청됩니다).

## 사용법

### 전송 번역 (내 채팅 번역)

1. `ValorantTranslator.exe` 실행
2. 출발 언어 / 도착 언어 / **전송 단축키** 선택 후 **적용** 클릭
3. 발로란트에서 **Enter** 눌러 채팅창 열기
4. 번역할 텍스트 입력
5. 전송 단축키(기본: **F9**) 누르기 → 텍스트가 번역된 내용으로 자동 교체
6. **Enter** 눌러 전송

### 읽기 번역 (상대방 채팅 번역)

1. 화면에 외국어 채팅이 보이는 상태에서 **읽기 단축키(기본: F10)** 누르기
2. 화면 왼쪽 하단 채팅 영역을 자동 캡처 → OCR → 한국어 번역
3. 결과가 반투명 오버레이 창으로 표시됨 (10초 후 자동 숨김)
4. 오버레이 상단 바를 드래그해 위치 이동 가능, `×`로 즉시 닫기

> **첫 실행 시 주의:** 읽기 번역 최초 사용 때 OCR 모델(약 300–500 MB)을 자동 다운로드합니다. 이후에는 캐시되어 즉시 동작합니다.

## 기능

- **자동 언어 감지** 포함 15개 언어 지원
- 출발/도착 언어를 GUI에서 실시간 변경
- 전송/읽기 단축키 각각 독립 설정 (F5~F12, Ctrl+1~5, Ctrl/Alt+F5~F10 등)
- 해상도 프리셋으로 채팅 영역 자동 조정 (1080p / 1440p / 4K)
- 번역 기록 실시간 표시
- 발로란트 전체화면(테두리 없는 창 모드)에서 정상 동작

## 지원 언어

자동 감지, 한국어, 일본어, 영어, 중국어(간체/번체), 프랑스어, 독일어, 스페인어, 러시아어, 포르투갈어, 이탈리아어, 베트남어, 태국어, 인도네시아어

## 소스에서 직접 실행

### 요구 사항

- Python 3.8+
- Windows 10/11

### 설치 및 실행

```bash
pip install -r requirements.txt
python main.py
```

### 의존성

| 패키지 | 용도 |
|--------|------|
| `deep-translator` | Google 번역 |
| `keyboard` | 전역 단축키 후킹 |
| `pyperclip` | 클립보드 제어 |
| `mss` | 화면 캡처 |
| `easyocr` | 다국어 OCR |
| `Pillow` | 이미지 전처리 |
| `numpy` | OCR 입력 변환 |

## 설정 (`config.py`)

| 항목 | 기본값 | 설명 |
|------|--------|------|
| `SOURCE_LANG` | `'auto'` | 전송 번역 입력 언어 |
| `TARGET_LANG` | `'ja'` | 전송 번역 목표 언어 |
| `HOTKEY` | `'f9'` | 전송 번역 단축키 |
| `READ_HOTKEY` | `'f10'` | 읽기 번역 단축키 |
| `CHAT_REGION` | `(5, 820, 430, 160)` | 채팅 캡처 영역 (x, y, w, h) — 1080p 기준 |
| `OVERLAY_POS` | `(5, 615)` | 번역 오버레이 초기 위치 |

> GUI에서 실행 중에도 변경 가능합니다. 해상도가 다를 경우 GUI의 **해상도** 드롭다운을 조정하세요.
