# Valorant 채팅 번역기

발로란트 채팅창에 한국어를 입력한 뒤 단축키를 누르면 일본어로 자동 교체해주는 툴입니다.

## 사용법

1. `python main.py` 실행 (백그라운드에서 동작)
2. 발로란트에서 **Enter** 눌러 채팅창 열기
3. 한국어로 하고 싶은 말 입력 (예: `안녕하세요`)
4. **F9** 누르기 → 입력한 한국어가 일본어로 교체됨 (예: `こんにちは`)
5. **Enter** 눌러 전송

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python main.py
```

## 설정 (`config.py`)

| 항목 | 기본값 | 설명 |
|------|--------|------|
| `SOURCE_LANG` | `'ko'` | 입력 언어 |
| `TARGET_LANG` | `'ja'` | 번역 목표 언어 |
| `HOTKEY` | `'f9'` | 번역 단축키 |

## 요구 사항

- Python 3.8+
- Windows 10/11
- 발로란트 설정에서 F9 키가 미사용 상태여야 함
