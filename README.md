# Valorant 채팅 번역기

발로란트 채팅창에 텍스트를 입력한 뒤 단축키를 누르면 원하는 언어로 자동 교체해주는 툴입니다.
Google Translate 기반으로 **API 키 없이** 무료로 사용할 수 있습니다.

## 다운로드 (exe)

별도 설치 없이 바로 실행하려면 [Releases](../../releases/latest) 페이지에서 `ValorantTranslator.exe`를 다운로드하세요.

> 관리자 권한이 필요합니다 (키보드 후킹을 위해 자동 요청됩니다).

## 사용법

1. `ValorantTranslator.exe` 실행
2. 출발 언어 / 도착 언어 / 단축키 선택 후 **적용** 클릭
3. 발로란트에서 **Enter** 눌러 채팅창 열기
4. 번역할 텍스트 입력
5. 단축키(기본: **F9**) 누르기 → 텍스트가 번역된 내용으로 자동 교체
6. **Enter** 눌러 전송

## 기능

- **자동 언어 감지** 포함 15개 언어 지원
- 출발/도착 언어를 GUI에서 실시간 변경 가능
- 단축키 드롭다운 선택 (F5~F12, Ctrl+1~5, Ctrl/Alt+F5~F10 등)
- 번역 기록 실시간 표시
- 발로란트 전체화면(테두리 없는 창 모드)에서 정상 동작

## 지원 언어

자동 감지, 한국어, 일본어, 영어, 중국어(간체/번체), 프랑스어, 독일어, 스페인어, 러시아어, 포르투갈어, 이탈리아어, 베트남어, 태국어, 인도네시아어

## 소스에서 직접 실행

### 요구 사항

- Python 3.8+
- Windows 10/11

### 설치 및 실행

```bash
pip install -r requirements.txt
python main.py
```

## 설정 (`config.py`)

| 항목 | 기본값 | 설명 |
|------|--------|------|
| `SOURCE_LANG` | `'auto'` | 입력 언어 (auto = 자동 감지) |
| `TARGET_LANG` | `'ja'` | 번역 목표 언어 |
| `HOTKEY` | `'f9'` | 번역 단축키 |

> GUI에서 실행 중에도 변경 가능합니다.
